# Micropython | Raspberry Pi Pico with DHT-11

## Preparation Hardware
- Raspberry Pi Pico
- DHT-11
- 3 Cable Female to Female

## Schematic 

| DHT 11 | Raspberry Pi Pico |
|--------|-------------------|
| GND    | GND               |
| VCC    | VYSY (5 v)        |
| DATA   | GP3               |
