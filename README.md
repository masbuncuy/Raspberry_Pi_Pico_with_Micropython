# Micropython | Raspberry Pi Pico with Ultrasonic Sensor HCSR-04

## Preparation Hardware
- Raspberry Pi Pico
- Ultrasonic Sensor HCSR-04
- 4 Cable Female to Female

## Schematic 

| Ultrasonic Sensor HCSR-O4 | Raspberry Pi Pico |
|---------------------------|-------------------|
| GND                       | GND               |
| VCC                       | VYSY (5 v)        |
| TRIG                      | GP4               |
| ECHO                      | GP5               |
