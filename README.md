# Micropython | Raspberry Pi Pico with OLED SSD1306

## Preparation Hardware
- Raspberry Pi Pico
- OLED SSD1306
- 4 Cable Female to Female

## Schematic 

| OLED SSD 1306 | Raspberry Pi Pico |
|---------------|-------------------|
| GND           | GND               |
| VCC           | VBUS              |
| SCL           | GP1               |
| SDA           | GP0               |
